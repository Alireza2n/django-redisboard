=============================
    django-redisboard
=============================


Brief redis monitoring in django admin

Installation guide
==================

Add ``redisboard`` to ``INSTALLED_APPS``::

    INSTALLED_APPS += ("redisboard", ) 
    
Then you can add redis servers in the admin. You will see the stats in the changelist.
